Overview

A simple Java console application to manage students, record daily attendance, and generate attendance reports. Data is stored using CSV files (students.csv, attendance.csv).

Features

Add & view students

Mark attendance (Present/Absent)

View attendance percentage

View daily attendance sheet

CSV file-based data storage

Technologies Used

Java (JDK 8+)

File Handling (CSV)

LocalDate API

Collections Framework

How to Install & Run
javac *.java
java Main


Files will auto-create:

students.csv

attendance.csv
